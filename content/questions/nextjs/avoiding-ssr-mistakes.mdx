# Deep Answer

### Interview Answer (Aise bolna hai):
Main ye mistakes consciously avoid karta hoon: SSR ko default solution samajhna, server pe unnecessary kaam karna, aur hydration issues ko ignore karna.

Main hamesha ye yaad rakhta hoon ki SSR ka goal user experience hai, not architectural purity. Agar SSR complexity zyada hai aur benefit kam, to main simpler approach choose karta hoon. Senior engineer ka kaam hota hai balance maintain karna, not over-engineering.
