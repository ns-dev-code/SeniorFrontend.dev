# Deep Answer

### Interview Answer (Aise bolna hai):
i18n SSR ke saath tab easy hota hai jab locale detection early ho. Cookies, headers, ya URL structure se locale detect karta hoon. SSR ke time correct language content render hota hai, taaki SEO aur UX dono benefit mile.

Client side pe language switch ke time hydration issues na aaye, isliye main locale state carefully sync karta hoon.
