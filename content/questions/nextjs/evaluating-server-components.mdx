# Deep Answer

### Interview Answer (Aise bolna hai):
Server Components mujhe conceptually powerful lagte hain, par main unko blindly adopt nahi karta. Main pehle dekhta hoon kya real benefit mil raha hai â€” bundle size reduce ho raha hai ya not, data fetching simpler ho raha hai ya not.

Agar component mostly data-driven hai aur interaction kam hai, to server component fit hota hai. Par agar component heavy interactive hai, to client component hi better hota hai. Mix approach best lagta hai.
