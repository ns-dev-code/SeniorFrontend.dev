# Deep Answer

### Interview Answer (Aise bolna hai):
Main sabse pehle ye clarity rakhta hoon ki kaunsa env variable server-only hai aur kaunsa client visible. Client visible variables accidentally expose ho jaate hain, aur wo security risk ban jata hai.

I prefer server-only secrets for tokens/keys. Client ko sirf public config milna chahiye. Aur env validation important hai — missing env variable ka issue runtime me nikalna worst hota hai. Main startup pe validate karna prefer karta hoon so issues early catch ho jaate hain.
