# Deep Answer

### Interview Answer (Aise bolna hai):
Caching ko main layers me sochta hoon: browser cache, CDN cache, server cache. SSG me page already static hota hai, so caching naturally best hoti hai.

SSR me caching tricky hoti hai because personalization ho sakta hai. Isliye main decide karta hoon: page truly personalized hai ya sirf “frequently updated” hai. Agar personalized nahi hai, SSR response caching possible hoti hai. Agar personalized hai, to at least API response caching or partial caching possible hoti hai.

Main hamesha ye ensure karta hoon ki caching strategy data correctness break na kare. Speed achhi hai, par wrong price/availability dikhana sabse badi problem hai.
