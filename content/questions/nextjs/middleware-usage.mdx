# Deep Answer

### Interview Answer (Aise bolna hai):
Middleware useful hota hai when you want request-time decisions quickly â€” auth redirects, geo-based routing, A/B tests, personalization gates.

But middleware ka risk ye hai ki wo har request pe run hota hai. Agar middleware heavy logic karne lage, to performance hit hogi. Isliye main middleware ko minimal rakhta hoon: quick checks, lightweight redirects. Heavy computation ya multiple API calls middleware me avoid karta hoon.
