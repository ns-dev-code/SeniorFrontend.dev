# Deep Answer

### Interview Answer (Aise bolna hai):
Analytics ka sabse bada rule mere liye ye hai: tracking kabhi UX ya performance ko hurt nahi kare.

SSR ke time main analytics scripts inject nahi karta in blocking way. Client side pe async load hota hai. Events tab fire hote hain jab UI actually render ho jaaye. Main ye bhi ensure karta hoon ki SSR render duplicate events na trigger kare.
