# Deep Answer

### Interview Answer (Aise bolna hai):
SSR me API failure ka impact direct user pe padta hai, isliye yahan error handling aur zyada important ho jaati hai. Main hamesha pehle decide karta hoon: Is API ke bina page completely useless ho jaayega ya partial render possible hai?

Agar critical data fail ho jaata hai, to main SSR pe hi safe fallback ya error page dikhata hoon. Par agar non-critical data fail hua hai, to main page ko render hone deta hoon with graceful empty state.
