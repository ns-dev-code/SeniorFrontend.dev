# Deep Answer

### Interview Answer (Aise bolna hai):
Ye kaafi practical issue hai. Agar tum everything split kar doge, initial page pe too many chunks load honge. Agar kuch bhi split nahi karoge, bundle fat ho jayega.

Main usually split karta hoon those components jo rarely used hain or heavy hain: charts, editors, modals, personalization widgets. Aur above-the-fold content ko stable rakhta hoon. SSR ke saath main yeh ensure karta hoon ki first render ke liye minimum assets required ho.
