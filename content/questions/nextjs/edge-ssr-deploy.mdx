# Deep Answer

### Interview Answer (Aise bolna hai):
Edge SSR exciting hai kyunki latency kam hoti hai. Par main hamesha check karta hoon ki logic edge-compatible hai ya nahi. Heavy computation, large libraries edge pe risky ho sakti hain.

Main edge ko mostly lightweight personalization aur routing ke liye use karta hoon. Right tool for right job — edge sab cheez ka solution nahi hai.
