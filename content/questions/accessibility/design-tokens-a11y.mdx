# Deep Answer

### Interview Answer (Aise bolna hai):
Design tokens consistency enforce karte hain. Agar colors, spacing aur typography tokens se aa rahe hain, to accessibility ek jagah control ho jaati hai.

Accessible color palette agar tokenized hai, to koi developer accidentally low-contrast color use nahi kar sakta.

Tokens accessibility ko scalable bana dete hain across teams.
