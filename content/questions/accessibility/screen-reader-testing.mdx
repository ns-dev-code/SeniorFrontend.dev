# Deep Answer

### Interview Answer (Aise bolna hai):
Main daily screen reader user nahi hoon, par critical flows ke liye main zaroor test karta hoon. Mostly VoiceOver ya NVDA use kiya hai.

Isse mujhe ye samajh aata hai ki screen reader actually kya bol raha hai, sirf kya dikh raha hai ye nahi.

Kai baar visually sab sahi hota hai, par screen reader confusing announcements deta hai. Wahin se real accessibility issues samne aate hain.
