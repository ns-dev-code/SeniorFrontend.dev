# Deep Answer

### Interview Answer (Aise bolna hai):
Honestly bolun to accessibility aur speed ko main opposite cheezein nahi maanta. Zyada time tab lagta hai jab accessibility ko baad me fix karna padta hai.

Agar start se semantic HTML aur correct patterns follow ho rahe hain, to accessibility "extra kaam" lagti hi nahi.

Jab timeline tight hoti hai, main at least ye ensure karta hoon ki core flows accessible hon. Perfect score se zyada important hota hai usable experience.
