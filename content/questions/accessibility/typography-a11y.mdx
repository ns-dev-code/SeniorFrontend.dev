# Deep Answer

### Interview Answer (Aise bolna hai):
Typography accessibility kaafi silent impact daalti hai. Agar font too small hai ya line-height cramped hai, to readability suffer karti hai.

Design system me main minimum font sizes aur proper line-heights define karta hoon. Developers ko arbitrary values use karne se rokta hoon.

Readable typography ka benefit sirf accessibility users ko nahi, sabko milta hai.
