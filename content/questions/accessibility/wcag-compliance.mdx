# Deep Answer

### Interview Answer (Aise bolna hai):
Main WCAG ko ratta maar ke follow nahi karta. Main usko common-sense usability ke angle se dekhta hoon.

Jaise keyboard se pura flow possible hona chahiye, focus clearly visible hona chahiye, text readable hona chahiye, aur forms understandable hone chahiye. Agar ye basics solid hain, to mostly AA compliance automatically cover ho jaata hai.

Automation tools help karte hain, par main uspe blind depend nahi karta. Main hamesha manually verify karta hoon ki real user ke liye experience smooth lag raha hai ya nahi.
