# Deep Answer

### Interview Answer (Aise bolna hai):
Accessibility regressions silently aati hain. Isliye main basic automated checks CI me rakhta hoon.

Par critical components ke liye main periodic manual testing bhi karta hoon. Especially jab major UI changes aaye ho.

Accessibility ek one-time task nahi hai, ye continuous process hai.
