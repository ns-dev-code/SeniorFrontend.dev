# Deep Answer

### Interview Answer (Aise bolna hai):
Honestly bolun to accessibility mere liye WCAG document ya audit report nahi hai. Mere liye accessibility ka matlab hai ki agar koi user mouse use nahi kar sakta, ya screen clearly nahi dekh sakta, tab bhi wo product ka main kaam bina struggle ke kar paaye.

Maine ye cheez tab seriously samjhi jab maine apni hi site ko sirf keyboard se use karke dekha. Us moment pe realize hua ki hum jo normal flow assume karte hain, wo sabke liye normal nahi hota.

Isliye main accessibility ko "last checklist item" nahi rakhta. Main usko UI aur component design ka natural part bana deta hoon.
