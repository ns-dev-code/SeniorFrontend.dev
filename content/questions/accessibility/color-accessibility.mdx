# Deep Answer

### Interview Answer (Aise bolna hai):
Contrast ratios baseline hain, par accessibility sirf numbers tak limited nahi hai. Color ke through information convey karna risky hota hai.

Jaise sirf red color se error dikhana — color-blind users ke liye problem ho sakta hai. Isliye main color ke saath text, icon ya message bhi add karta hoon.

Mera approach ye hota hai ki agar color remove kar diya jaaye, tab bhi meaning clear rahe.
