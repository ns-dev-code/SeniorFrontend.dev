# Deep Answer

### Interview Answer (Aise bolna hai):
Scale pe accessibility tab break hoti hai jab ownership unclear hoti hai. Shuru me sab theek hota hai, par jaise-jaise features badhte hain, small shortcuts lene lagte hain.

Isliye main accessibility ko individual developer ke upar nahi chhodta. Main design system ke through guardrails set karta hoon.

Agar base components accessible hain, to 70–80% problems automatically solve ho jaati hain. Baaki ke liye PR reviews aur basic automated checks ka role hota hai. Accessibility ko scale pe maintain karne ka secret hai — defaults ko strong banana.
