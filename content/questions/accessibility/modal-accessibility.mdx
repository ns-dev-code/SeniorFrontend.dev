# Deep Answer

### Interview Answer (Aise bolna hai):
Modal accessibility ek tarah se system ka stress test hota hai. Yahin se pata chalta hai design system kitna mature hai.

Modal open hote hi focus uske andar trap hona chahiye. Background content keyboard se accessible nahi hona chahiye.

Escape key se modal close hona chahiye, aur close hone ke baad focus wapas wahi jaana chahiye jahan se modal open hua tha. Agar ye flow smooth hai, to modal accessible bhi hota hai aur UX bhi strong hota hai.
