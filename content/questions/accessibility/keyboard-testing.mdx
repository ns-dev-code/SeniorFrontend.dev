# Deep Answer

### Interview Answer (Aise bolna hai):
Main koi fancy setup nahi karta. Main mouse side me rakh deta hoon aur sirf keyboard se site chalata hoon.

Tab key se navigation logical lag rahi hai ya nahi, focus visible hai ya nahi, Enter aur Space expected jagah kaam kar rahe hain ya nahi — ye sab main manually check karta hoon.

Modals, dropdowns aur forms yahin pe sabse zyada issues dikhate hain. Agar keyboard flow smooth hai, to accessibility kaafi strong hoti hai.
