# Deep Answer

### Interview Answer (Aise bolna hai):
Sabse common mistake hoti hai missing labels. Placeholder ko label samajh liya jaata hai, jo screen readers ke liye kaam nahi karta.

Dusri badi problem hoti hai error handling. Error visually dikh raha hota hai, par screen reader ko announce hi nahi hota.

Isliye main ensure karta hoon ki har input ka proper label ho, aur error message input ke saath programmatically connected ho.
