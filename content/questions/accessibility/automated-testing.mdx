# Deep Answer

### Interview Answer (Aise bolna hai):
Main automation ko starting point maanta hoon, final answer nahi. Lighthouse aur axe basic issues pakad lete hain.

Par real accessibility problems tab nikalti hain jab tum manually keyboard aur screen reader se test karte ho. Isliye main automation + manual testing dono combine karta hoon.

Automation regression rokti hai, manual testing quality improve karti hai.
