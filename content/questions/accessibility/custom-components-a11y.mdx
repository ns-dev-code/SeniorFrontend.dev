# Deep Answer

### Interview Answer (Aise bolna hai):
Custom components accessibility ka sabse risky area hota hai. Tabs aur accordions visually simple lagte hain, par behavior-wise complex hote hain.

Main pehle native behavior samajhta hoon — keyboard navigation, focus order, announcements. Uske baad hi custom implementation karta hoon.

ARIA roles tab use karta hoon jab required ho, par uske saath keyboard behavior bhi replicate karta hoon. Sirf role add kar dena kaafi nahi hota.
