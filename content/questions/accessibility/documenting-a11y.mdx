# Deep Answer

### Interview Answer (Aise bolna hai):
Documentation accessibility ke liye bahut important hoti hai. Sirf code accessible hona kaafi nahi, usage bhi clear hona chahiye.

Main design system docs me clearly likhta hoon ki component ka intended usage kya hai, keyboard behavior kya hai, aur kya avoid karna chahiye.

Isse new developers galat patterns follow nahi karte.
