# Deep Answer

### Interview Answer (Aise bolna hai):
Large lists me sabse pehli galti hoti hai sab kuch DOM me daal dena. Initially sab smooth lagta hai, par thoda scroll karte hi UI lag karne lagta hai.

Main hamesha pagination ya virtualization consider karta hoon. User ko ek baar me 500 items dikhane ka koi real use nahi hota. Aur main ye bhi ensure karta hoon ki list item component lightweight ho. Heavy child components list performance ko quietly destroy kar dete hain.
