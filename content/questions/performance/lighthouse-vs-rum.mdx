# Deep Answer

### Interview Answer (Aise bolna hai):
Lighthouse ek achha starting point hai, par main uspe blindly trust nahi karta. Ye lab conditions me run hota hai, jo real users ko fully represent nahi karta.

Agar Lighthouse green hai par users complain kar rahe hain, to main real user metrics dekhta hoon. Mobile devices, slow networks — yahin asli picture milti hai. Main Lighthouse ko debugging tool ki tarah use karta hoon, truth ki tarah nahi.
