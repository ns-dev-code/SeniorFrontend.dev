# Deep Answer

### Interview Answer (Aise bolna hai):
CLS usually tab nikalta hai jab UI thoda “unstable” ho. Aur ye cheez users ko bahut irritate karti hai, even if site fast ho.

Maine sabse zyada CLS issues images aur fonts ki wajah se dekhe hain. Image ke liye height/width reserve nahi ki, to page jump karta hai. Font late load hua, to text shift ho gaya.

Fix usually boring hota hai, but effective. Space reserve karo, fonts ko preload karo, aur dynamic content ke liye skeleton ya placeholder rakho. Ek aur common CLS source hota hai marketing banners jo late inject ho jaate hain. Aise cases me main hamesha space pehle se reserve karta hoon, taaki layout stable rahe.
