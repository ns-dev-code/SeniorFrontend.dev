# Deep Answer

### Interview Answer (Aise bolna hai):
Haan, hua hai. Example ke liye, focus outlines ya additional ARIA attributes thoda extra DOM add karte hain. Par main accessibility ko performance ke naam pe kabhi compromise nahi karta. Impact usually negligible hota hai, aur user impact bahut zyada.

Agar performance aur accessibility me conflict lag raha hai, to usually problem design ya implementation me hoti hai, accessibility me nahi.
