# Deep Answer

### Interview Answer (Aise bolna hai):
SSR performance ko improve bhi kar sakta hai aur degrade bhi. Sab depend karta hai implementation pe.

Agar SSR me heavy data fetch ya computation hai, to TTFB badh jaata hai. User ko blank screen zyada time dikhta hai. Isliye main SSR ko lightweight rakhta hoon. Jo cheez first view ke liye critical hai sirf wahi SSR me aati hai.
