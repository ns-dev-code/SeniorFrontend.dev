# Deep Answer

### Interview Answer (Aise bolna hai):
Main performance ko kabhi guess se approach nahi karta. Initial days me maine ye galti ki hai ki bina measure kiye optimize kar diya, aur baad me pata chala ki real problem kahin aur thi.

Ab mera approach simple hai. Sabse pehle main measure karta hoon — Lighthouse, DevTools, ya real user metrics agar available ho. Uske baad main identify karta hoon ki problem actually rendering me hai, network me hai, ya JavaScript execution me.

Performance issues usually ek jagah se nahi aate. Kabhi heavy images hote hain, kabhi third-party scripts, kabhi unnecessary re-renders. Isliye pehle clarity laana important hota hai, phir solution natural nikalta hai.
