# Deep Answer

### Interview Answer (Aise bolna hai):
Mobile devices slower CPU aur network pe hote hain. Jo cheez desktop pe smooth lagti hai, mobile pe immediately expose ho jaati hai. Isliye main performance testing hamesha mobile-first karta hoon. Agar mobile pe smooth hai, to desktop automatically fine hota hai. Mobile performance mere liye “truth detector” jaisa hai.
