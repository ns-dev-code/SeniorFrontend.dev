# Deep Answer

### Interview Answer (Aise bolna hai):
Sabse misleading cheez hoti hai “local machine”. Local pe sab fast lagta hai, par prod environment bilkul different hota hai.

Isliye main real network conditions simulate karta hoon. Aur prod data ke saath test karta hoon, dummy data ke saath nahi. Performance hamesha context me samajhni padti hai.
