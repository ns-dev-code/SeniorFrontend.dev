# Deep Answer

### Interview Answer (Aise bolna hai):
Kyuki real users mobile pe hote hain. Desktop pe sab kuch fast lagta hai, par mobile pe same code struggle karta hai.

Main hamesha mobile throttling ke saath test karta hoon. Agar mobile pe smooth hai, to desktop automatically smooth ho jaata hai. Performance optimize karte waqt main worst-case user ko target karta hoon, best-case ko nahi.
