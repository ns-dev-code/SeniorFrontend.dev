# Deep Answer

### Interview Answer (Aise bolna hai):
Regression tab hota hai jahan ek feature ke saath performance silently gir jaati hai. Aur ye tab pakad me aata hai jab damage ho chuka hota hai.

Main performance ko review process ka part banata hoon. PR me heavy changes aaye to bundle size aur load impact check karta hoon. Jahan possible ho, main performance budgets set karta hoon. Isse team ko pata hota hai ki “limit cross ho rahi hai”.
