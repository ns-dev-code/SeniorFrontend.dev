# Deep Answer

### Interview Answer (Aise bolna hai):
Ek project me Lighthouse score kaafi decent tha, aur internally sabko lag raha tha performance theek hai. Par production me users complain kar rahe the ki “site lag karti hai”.

Jab maine thoda deep dive kiya, to pata chala issue React ya rendering ka nahi tha. Problem thi ek third-party personalization script jo page load ke baad heavy computation kar rahi thi. Local testing me wo noticeable nahi thi, par low-end mobile devices pe main thread block ho jaata tha.

Uske baad maine script ko interaction ke baad load karwaya, aur kuch logic debounce kiya. Lesson ye tha ki performance sirf initial load ka game nahi hai, interaction ke baad ka behavior bhi utna hi important hai.
