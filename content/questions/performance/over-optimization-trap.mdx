# Deep Answer

### Interview Answer (Aise bolna hai):
Ek baar maine over-memoization ki galti ki. Almost har component me useMemo aur useCallback daal diya, bina proper profiling ke. Result ye hua ki code complexity badh gayi, debugging mushkil ho gayi, aur performance gain negligible tha.
